# 🧩 Jogo dos 8 Números (8-Puzzle)

Um clássico jogo de raciocínio lógico construído inteiramente com HTML, CSS e JavaScript (Vanilla). O objetivo do jogo é ordenar as peças numeradas de 1 a 8, utilizando o espaço vazio para deslizar os blocos.

<img width="1917" height="908" alt="image" src="https://github.com/user-attachments/assets/904ba7df-7924-44f8-8c17-867dc7341433" />

## 🚀 Funcionalidades

* **Movimentação intuitiva:** Clique em qualquer peça adjacente ao espaço vazio para movê-la.
* **Embaralhamento seguro:** O botão "Embaralhar" simula movimentos reais invertidos, garantindo que o tabuleiro gerado sempre tenha uma solução matemática possível.
* **Detector de vitória:** O jogo reconhece automaticamente quando as peças estão na ordem correta e exibe uma mensagem de sucesso.
* **Gabarito:** Um botão de "Mostrar Solução" para resetar o tabuleiro ao estado inicial.

## 🛠️ Tecnologias Utilizadas

* HTML5 (Estrutura)
* CSS3 (Estilização, Grid Layout e Transições)
* JavaScript (Lógica de estado, matriz e manipulação do DOM)

## 🎮 Como jogar localmente

1. Faça o clone deste repositório:
   `git clone https://github.com/William3302/8-Puzzle`
2. Abra a pasta do projeto.
3. Dê um duplo clique no arquivo `index.html` para abri-lo em seu navegador padrão.
